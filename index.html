ABOUT

Who I am & how I think

I focus on clear structure, functional design and a calm black & white visual language. My projects combine frontend, backend, databases and, lately, local AI tools that improve how I understand and build applications.

Profile

I work across the full stack — from frontend (HTML, CSS, JS, UI/UX) to backend (PHP, Node.js) and database design using MySQL and structured data models.

My aim is always the same: make the project clean, organized and scalable. No noise, no random elements — everything has a purpose.

How I work

  • Structure first

    Sections, logic and components are planned before design begins.

  • Frontend ↔ Backend flow

    I always think in terms of complete flow: UI → logic → database → response.

  • Local AI tools

    I use Ollama and my own AI assistant Nexora to analyse code, structure ideas and speed up development safely and offline.

  • Subtle animations

    Clean micro-interactions that support the design instead of distracting from it.
